# React - Typescript - Beer APP

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Features

- Display beer list
- Display single beer
- Filter by beer name
- Filter by Alcohol content (20%)
- Sort by % Alcohol
- Sort by Name
